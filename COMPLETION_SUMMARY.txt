🔄 HOW IT WORKS
════════════════════════════════════════════════════════════════════════════════

  Theme Toggle Flow:
    1. User clicks toggle in OwnerNavbar
    2. OwnerNavbar saves theme to localStorage
    3. OwnerNavbar dispatches window.themeChange event
    4. Each page listens for event
    5. Page updates isDark state
    6. Page re-renders with new className
    7. CSS applies appropriate colors

  Event Communication:
    • Event Name:    window.themeChange
    • Source:        OwnerNavbar.jsx
    • Listeners:     All 3 owner pages
    • Persistence:   localStorage (key: 'theme')
    • No reload:     Real-time updates
